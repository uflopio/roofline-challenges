Apply the GELU activation function element-wise.

Given vector $\underline{x}$ of length $N$, produce vector $\underline{y}$ of length $N$ such that

$$
y_i = 0.5 \cdot x_i \cdot \left( 1 + \tanh\left( \sqrt{\tfrac{2}{\pi}} \left( x_i + 0.044715 \cdot x_i^3 \right) \right) \right) \quad \text{for } i = 0, 1, \dots, N - 1.
$$

Use the *approximate* (tanh-based) form above, not the exact erf-based GELU.

## Input

- One vector $\underline{x}$ of length $N$.

## Output

- One vector $\underline{y}$ of length $N$.

## Sizes

- $N = 2^{20}$
- $N = 2^{22}$
- $N = 2^{24}$
- $N = 2^{26}$
- $N = 2^{28}$
