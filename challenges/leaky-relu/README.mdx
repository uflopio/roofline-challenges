Perform the Leaky ReLU (Leaky Rectified Linear Unit) activation function on an input matrix:
$$
C[i][j] = \max(\alpha \cdot A[i][j], A[i][j])
$$
where $\alpha$ is a small positive constant (e.g. 0.01)

The Leaky ReLU function is defined as:
$$
f(x) = \begin{cases}
x & \text{if } x > 0 \\
\alpha x & \text{if } x \leq 0
\end{cases}
$$

## Input
- Matrix $A$ of size $M \times N$
- $\alpha$ value (slope for negative values)

## Output
- Matrix $C$ of size $M \times N$

## Sizes

- $M = 2^{12}, N = 2^{12}$
- $M = 6144, N = 2^{12}$
