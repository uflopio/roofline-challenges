Perform 4D tensor-matrix multiplication of two tensors:
$$
C_{b,i}[j][k] = \sum_{l=0}^{L-1} A_{b,i}[j][l] \cdot B_{l,k}
$$

## Input
- Tensor $A$ of size $B \times I \times J \times L$
- Matrix $B$ of size $L \times K$

## Output
- Tensor $C$ of size $B \times I \times J \times K$

## Sizes

- $B = 16, I = 2^{8}, J = 2^{9}, L = 2^{8}, K = 768$
- $B = 8, I = 128, J = 2^{8}, L = 128, K = 2^{9}$
- $B = 32, I = 64, J = 128, L = 64, K = 2^{8}$
- $B = 4, I = 32, J = 64, L = 32, K = 128$
