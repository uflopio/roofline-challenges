Perform 3D average pooling on an input tensor:
$$
\text{output}[i,j,k] = \frac{1}{k^3}\sum_{m=0}^{k-1}\sum_{n=0}^{k-1}\sum_{o=0}^{k-1} \text{input}[S \cdot i + m - P, S \cdot j + n - P, S \cdot k + o - P]
$$

The average pooling operation slides a window of size $k \times k \times k$ over the input tensor with stride $S$ and padding $P$, computing the average value within each window position.

## Input
- Matrix `input` of size $H \times W \times D$ (input tensor)
- `kernel_size` ($k$): Size of the pooling window
- `stride` ($S$): Step size between window positions
- `padding` ($P$): Number of zero-padding elements added on all sides

## Output
- Matrix `output` of size $H_{\text{out}} \times W_{\text{out}} \times D_{\text{out}}$ where:
  $$H_{\text{out}} = \left\lfloor\frac{H + 2P - k}{S} + 1\right\rfloor$$
  $$W_{\text{out}} = \left\lfloor\frac{W + 2P - k}{S} + 1\right\rfloor$$
  $$D_{\text{out}} = \left\lfloor\frac{D + 2P - k}{S} + 1\right\rfloor$$

## Sizes

- $KERNEL_SIZE = 192, STRIDE = 192, PADDING = 192, H = 5, W = 2, D = 2$
- $KERNEL_SIZE = 224, STRIDE = 224, PADDING = 224, H = 7, W = 3, D = 3$
- $KERNEL_SIZE = 2^{9}, STRIDE = 2^{9}, PADDING = 2^{9}, H = 3, W = 3, D = 1$
- $KERNEL_SIZE = 784, STRIDE = 784, PADDING = 784, H = 5, W = 2, D = 2$
- $KERNEL_SIZE = 2^{10}, STRIDE = 2^{10}, PADDING = 2^{10}, H = 3, W = 3, D = 1$
