Perform multiplication of a matrix and a vector:
$$
C[i] = \sum_{k=0}^{K-1} A_{i,k} \cdot B[k]
$$

## Input
- Matrix $A$ of size $M \times K$
- Vector $B$ of size $K \times 1$

## Output
- Vector $C = AB$ of size $M \times 1$

## Sizes

- $M = 2^{12}, K = 2^{12}$
- $M = 6144, K = 2^{12}$
- $M = 7168, K = 2^{12}$
- $M = 2^{13}, K = 2^{12}$
- $M = 9216, K = 2^{12}$
