Perform a 2D convolution followed by ReLU activation followed by HardSwish activation:

1. **2D Convolution**:
   $$C_{i,j} = \sum_{u=0}^{K_h-1} \sum_{v=0}^{K_w-1} I\left[i+u-\frac{K_h-1}{2}\right]\left[j+v-\frac{K_w-1}{2}\right] \cdot K_{u,v}$$

2. **ReLU Activation**:
   $$R_{i,j} = \max(0, C_{i,j})$$

3. **HardSwish Activation**:
   $$O_{i,j} = R_{i,j} \cdot \frac{\text{ReLU6}(R_{i,j} + 3)}{6}$$

where $\text{ReLU6}(x) = \min(6, \max(0, x))$.

## Input
- `image` of size $H \times W$
- `kernel` of size $K_h \times K_w$ (both dimensions must be odd)

## Output
- `output` of size $H \times W$

## Sizes

- $H = 2^{9}, W = 2^{9}, Kh = 3, Kw = 3$
- $H = 2^{10}, W = 2^{10}, Kh = 5, Kw = 5$
- $H = 2^{11}, W = 2^{11}, Kh = 7, Kw = 7$
- $H = 2^{9}, W = 2^{9}, Kh = 9, Kw = 9$
- $H = 2^{10}, W = 2^{10}, Kh = 11, Kw = 11$
- $H = 2^{11}, W = 2^{11}, Kh = 13, Kw = 13$
