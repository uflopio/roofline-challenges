Perform 2D convolution between an input image and a kernel:
$$
C[i,j] = \sum_{k=0}^{K_h-1}\sum_{l=0}^{K_w-1} A[i+k,j+l] \cdot B[k,l]
$$

The convolution operation slides the 2D kernel over the input image, computing the sum of element-wise multiplications at each position. Zero padding is used at the boundaries.

## Input
- Matrix $A$ of size $H \times W$ (input image)
- Matrix $B$ of size $K_h \times K_w$ (convolution kernel)
- Both $K_h$ and $K_w$ are odd and smaller than $H$ and $W$ respectively

## Output
- Matrix $C$ of size $H \times W$ (convolved image)

## Sizes

- $H = 2^{12}, W = 2^{12}, Kh = 9, Kw = 9$
- $H = 2^{13}, W = 2^{13}, Kh = 11, Kw = 11$
- $H = 2^{14}, W = 2^{14}, Kh = 13, Kw = 13$
- $H = 2^{10}, W = 2^{10}, Kh = 31, Kw = 31$
- $H = 2^{11}, W = 2^{11}, Kh = 63, Kw = 63$
- $H = 2^{12}, W = 2^{12}, Kh = 127, Kw = 127$
