Perform multiplication of two symmetric matrices:
$$
C_{i,j} = \sum_{k=0}^{N-1} A_{i,k} \cdot B_{k,j}
$$

## Input
- $A$ is a symmetric matrix of size $N \times N$
- $B$ is a symmetric matrix of size $N \times N$

## Output
- Matrix $C = AB$ of size $N \times N$

## Sizes

- 4096x4096
- 6144x6144
- 7168x7168
- 8192x8192
- 9216x9216
