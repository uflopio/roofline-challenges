Perform 2D max pooling on an input tensor:
$$
\text{output}[i,j] = \max_{m=0,n=0}^{k-1,k-1} \text{input}[S \cdot i + D \cdot m - P, S \cdot j + D \cdot n - P]
$$

The max pooling operation slides a window of size $k \times k$ over the input tensor with stride $S$, dilation $D$, and padding $P$, computing the maximum value within each window position.

## Input
- Matrix `input` of size $H \times W$ (input tensor)
- `kernel_size` ($k$): Size of the pooling window
- `stride` ($S$): Step size between window positions
- `padding` ($P$): Number of zero-padding elements added on all sides
- `dilation` ($D$): Spacing between kernel elements

## Output
- Matrix `output` of size $H_{\text{out}} \times W_{\text{out}}$ where:
  $$H_{\text{out}} = \left\lfloor\frac{H + 2P - D(k-1) - 1}{S}\right\rfloor + 1$$
  $$W_{\text{out}} = \left\lfloor\frac{W + 2P - D(k-1) - 1}{S}\right\rfloor + 1$$

## Sizes

- $KERNEL_SIZE = 2^{12}, STRIDE = 2^{12}, PADDING = 2, DILATION = 2, H = 0, W = 1$
- $KERNEL_SIZE = 2^{13}, STRIDE = 2^{13}, PADDING = 3, DILATION = 2, H = 1, W = 1$
- $KERNEL_SIZE = 2^{14}, STRIDE = 2^{14}, PADDING = 4, DILATION = 4, H = 2, W = 1$
- $KERNEL_SIZE = 2^{10}, STRIDE = 2^{10}, PADDING = 3, DILATION = 3, H = 1, W = 2$
- $KERNEL_SIZE = 2^{11}, STRIDE = 2^{11}, PADDING = 5, DILATION = 2, H = 2, W = 1$
- $KERNEL_SIZE = 2^{12}, STRIDE = 2^{12}, PADDING = 7, DILATION = 3, H = 3, W = 1$
