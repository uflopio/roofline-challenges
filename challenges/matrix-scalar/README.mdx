Perform multiplication of a matrix with a scalar value:
$$
C_{i,j} = A_{i,j} \cdot s
$$
where $s$ is the scalar value.

## Input
- Matrix $A$ of size $\text{n} \times \text{n}$
- Scalar value $s$

## Output
- Matrix $C = s \cdot A$ of size $\text{n} \times \text{n}$

## Sizes

- $N = 2^{13}$
- $N = 9216$
