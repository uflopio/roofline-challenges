Implement scaled dot-product attention, a key component of transformer architectures:
$$
\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{E}}\right)V
$$

Where Q, K, and V are the query, key, and value matrices, and $E$ is the embedding dimension.

## Input
- Matrix $Q$, $K$, $V$ of size $(B \times H \times S \times E)$ (query, key, value)

## Output
- Matrix $\text{output}$ of size $(B \times H \times S \times E)$

## Sizes

- $B = 8, H = 16, S = 2^{9}, E = 64$
- $B = 16, H = 32, S = 2^{8}, E = 64$
- $B = 4, H = 32, S = 2^{10}, E = 32$
- $B = 32, H = 8, S = 2^{9}, E = 128$
- $B = 8, H = 16, S = 2^{9}, E = 2^{8}$
- $B = 8, H = 16, S = 2^{11}, E = 64$
