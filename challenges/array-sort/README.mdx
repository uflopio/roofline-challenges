Sort an array of integers in ascending order.

## Input
- Array $a$ of integers of size $n$

## Output
- The same array $a$ sorted in ascending order stored in $b$

## Sizes

- $N = 2^{14}$
- $N = 2^{15}$
- $N = 2^{16}$
- $N = 2^{17}$
- $N = 2^{18}$
