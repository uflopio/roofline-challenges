Compute the sum of all elements in a vector.

Given vector $\underline{a}$ of length $N$, produce the scalar $s$ such that

$$
s = \sum_{i=0}^{N-1} a_i.
$$

## Input

- One vector $\underline{a}$ of length $N$.

## Output

- One scalar $s$ such that $s = \sum_{i=0}^{N-1} a_i$.

## Sizes

- $N = 2^{20}$
- $N = 2^{22}$
- $N = 2^{24}$
- $N = 2^{25}$
- $N = 2^{26}$
