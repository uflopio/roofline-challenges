Perform 2D average pooling on an input tensor:
$$
\text{output}[i,j] = \frac{1}{k^2}\sum_{m=0}^{k-1}\sum_{n=0}^{k-1} \text{input}[S \cdot i + m - P, S \cdot j + n - P]
$$

The average pooling operation slides a window of size $k \times k$ over the input tensor with stride $S$ and padding $P$, computing the average value within each window position.

## Input
- Matrix `input` of size $H \times W$ (input tensor)
- `kernel_size` ($k$): Size of the pooling window
- `stride` ($S$): Step size between window positions
- `padding` ($P$): Number of zero-padding elements added on all sides

## Output
- Matrix `output` of size $H_{\text{out}} \times W_{\text{out}}$ where:
  $$H_{\text{out}} = \left\lfloor\frac{H + 2P - k}{S} + 1\right\rfloor$$
  $$W_{\text{out}} = \left\lfloor\frac{W + 2P - k}{S} + 1\right\rfloor$$

## Sizes

- $KERNEL_SIZE = 2^{13}, STRIDE = 2^{12}, PADDING = 2, H = 2, W = 0$
- $KERNEL_SIZE = 2^{13}, STRIDE = 2^{13}, PADDING = 3, H = 2, W = 1$
- $KERNEL_SIZE = 2^{14}, STRIDE = 2^{14}, PADDING = 4, H = 4, W = 2$
- $KERNEL_SIZE = 2^{13}, STRIDE = 2^{13}, PADDING = 3, H = 3, W = 1$
- $KERNEL_SIZE = 2^{11}, STRIDE = 2^{11}, PADDING = 5, H = 2, W = 2$
- $KERNEL_SIZE = 2^{12}, STRIDE = 2^{12}, PADDING = 7, H = 3, W = 3$
